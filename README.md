# Elevate Body Wellness

A production-ready Next.js 15 App Router website for a body wellness trainer.

## Tech Stack
- Next.js 15 (App Router)
- React 19
- TypeScript
- Tailwind CSS v4
- shadcn/ui
- Framer Motion
- Supabase (Database)
- Resend (Email)

## Features
- Fully responsive design with HSL custom theme
- Accessible Radix UI components
- Restrained Framer Motion animations (respects reduced-motion)
- Complete SEO setup with metadata and JSON-LD
- Server-side form handling and validation
- Graceful degradation (works without DB/Email keys)

## Getting Started

1. Clone the repo and install dependencies:
```bash
npm install
```

2. Copy the env file and add your keys (optional):
```bash
cp .env.local.example .env.local
```

3. Run the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
